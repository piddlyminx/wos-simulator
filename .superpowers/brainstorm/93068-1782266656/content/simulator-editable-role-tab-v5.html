What changed from v4

  • Each unit type is an editable row, not a summary tile.
  • Troop count is a visible input-like control.
  • Tier and hero read as dropdowns with chevrons.
  • Skills are explicit per hero, so rally skill 4 behavior has a place to live.
  • Stats are shown as a 3 × 4 matrix: each troop type has attack, defense, lethality, and health.
  • Joiners remain visible because they are required for normal rally runs.
